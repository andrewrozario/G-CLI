## 🧠 The Architecture of Action
G CLI is an autonomous, dependency-aware engineering engine. Unlike basic wrappers, G CLI orchestrates a cognitive relay between **Claude 3.5**, **Codex (GPT-4o)**, and **Gemini 1.5 Pro** to solve complex system-wide problems with surgical precision.

## 🚀 Instant Deployment

Install G CLI globally on your machine:

```bash
git clone git@github.com:andrewrozario/G-CLI.git
cd G-CLI
sh install.sh
```

## 🛠 Strategic Setup

Configure your frontier intelligence cores:
```bash
g config --claude YOUR_KEY --gemini YOUR_KEY --codex YOUR_KEY
```

Verify your system health and connection stability:
```bash
g doctor
```

## ⚡ Core Capabilities

### 🔀 Dependency-Aware Refactoring
G CLI builds a real-time dependency graph of your project to predict the ripple effects of any change.
```bash
g ./ "refactor the authentication middleware and update all dependencies"
```

### 🩹 Autonomous Self-Healing
G CLI automatically executes your test suites (`pytest`, `npm test`) after every operation. If breakage is detected, it **diagnoses and repairs the code autonomously** until stability is achieved.

### 👁️ Continuous Partnering (Watcher Mode)
Active your AI teammate to watch your work in real-time. G detects issues as you save and prepares surgical fixes proactively.
```bash
g watch --auto
```

### 🛡️ Industrial Safety Layer
- **High-Fidelity Diffs:** See exactly what changes before they are applied.
- **Auto-Backups:** Every file modification creates a `.bak` snapshot.
- **Git Integration:** Every successful improvement is automatically committed to your Git history.
- **Instant Undo:** Restore any file to its previous state with a single command.
