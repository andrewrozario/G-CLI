## ‚ö° Core Intelligence Modes

### 1. Surgical File Surgery (`g ask`)
Don't just generate code‚ÄĒoperate on it. G CLI automatically reads files, generates high-fidelity diffs, and waits for your confirmation before applying changes.
```bash

```

### 2. Dependency-Aware Refactoring (`g ./`)
G CLI understands the "ripple effect." It builds a project dependency graph to predict breakage and automatically updates all related files during a refactor.
```bash
g ./ "rename getUser function to fetchUser globally"
```

### 3. Continuous Partnering (`g watch`)
Activate G's "watcher" mode to have an AI teammate working alongside you. She watches your saves, detects issues, and prepares fixes before you even ask.
```bash
g watch --auto
```

### 4. Self-Healing Autonomy
G CLI automatically runs your test suites (`pytest`, `npm test`) after every change. If it breaks your code, it **detects the error and fixes itself** autonomously.
